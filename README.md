## 📖 Why I Built This

Every job posting I see for IT Support or System Administrator roles in Germany mentions the same thing: "Experience with SIEM tools like Splunk, Wazuh, or QRadar." The problem? How do you get SIEM experience when every entry-level job requires it, but no one will hire you without it?

I decided to break that cycle. If I couldn't get access to a corporate Security Operations Center, I'd build my own.

This is the story of how I went from reading about security monitoring to actually running a working SIEM that collects real security events from a Windows endpoint. It took about 6 hours, a lot of troubleshooting, and more coffee than I'd like to admit. But by the end, I had infrastructure that does exactly what enterprise SOC teams use to detect threats.

**The Goal:** Build a working Wazuh SIEM that monitors a Windows 11 endpoint and collects security events in real-time.

**What I learned along the way:** More than I expected.
